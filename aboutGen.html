About Jeopardy!

This Jeopardy question Generator was created to sort the countless seasons of Jeopardy in one easy place. You can search by date, category, or difficulty to find those specific questions. You can search without any input and a random question will come up.

This website's data was drawn from the restful API, http://jservice.io. This API provided questions asked on the show along with ways to sort through it.

Below is a link to my github with more information on creating this application and its features: https://github.com/lizandrof273/Jeopardy-Questions-App
